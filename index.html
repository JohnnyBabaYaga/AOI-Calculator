☁️ 雲端同步設定
透過 GitHub 讓不同電腦共同維護「專案評估紀錄」與「相機／鏡頭資料庫」。在每台電腦填入相同設定即可同步。
需具備該 repo 的 Contents 讀寫權限。權杖只存在本機瀏覽器，不會寫進 HTML 或上傳。
⚠ 資安提醒：資料會離開本機、存到 GitHub。請確認符合 AUO 資安規範，並使用 private（私有） repo。同步採「最後儲存為準」，多人共用建議先「從雲端讀取」再編輯。
